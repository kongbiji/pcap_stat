# pcap_stat
Wireshark packet statistics program

It is output similarly to Wireshark's Statistics > Conversations and Endpoints".

# How to use
```
git clone https://github.com/kongbiji/pcap_stat
```

```
cd pcap_stat
make
./pcap_sata test.pcap
```
